## 📌 Overview
A responsive **Sign-Up Form** with built-in HTML5 and JavaScript validation.  
It ensures users enter valid data and provides real-time visual feedback.

---

## ✨ Features
- Form validation using HTML5
- Password match verification
- Dynamic success & error messages
- Clean UI with responsive layout
- Console-based data handling

## 🧪 Validation Rules
- Name: 3–100 characters  
- Phone: `+123456789` format  
- Password: 8+ chars, uppercase, lowercase, number
